# 关键概念（Key Concepts）

| 概念 | 定义 | 证据 |
|---|---|---|
| DevBooks Skills | 以角色或任务为粒度的工作流能力模块 | `skills/` |
| 协议发现层 | 通过 `.devbooks/config.yaml` 或协议文件解析目录映射 | `scripts/config-discovery.sh` |
| DevBooks 集成 | 将 DevBooks 协议映射到 dev-playbooks 目录结构 | `setup/dev-playbooks/README.md` |
| MCP 配置 | Claude Code 与 Codex CLI 的 MCP 服务器配置方式 | `mcp/mcp_claude.md`，`mcp/mcp_codex.md` |
| 自动化守门 | CI 模板提供的架构合规检查 | `templates/ci/README.md` |
