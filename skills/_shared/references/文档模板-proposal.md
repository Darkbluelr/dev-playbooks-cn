# Proposal 文档模板

> 本模板定义了 proposal.md 的标准结构，确保提案清晰、可审查、可落地。

## 文件位置

```
<change-root>/<change-id>/proposal.md
```

## 使用指南

### 1. 结论先行的作用

- **30秒快速理解**：让用户立即知道"会导致什么，不会导致什么"
- **降低认知负担**：用大白话，不用专业术语
- **建立预期**：明确说明变化范围

### 2. 需求对齐的作用

- **角色定位**：帮助用户明确自己的角色和需求
- **挖掘隐藏需求**：通过问题引导用户思考可能忽略的需求
- **推荐方案**：基于角色给出推荐方案

### 3. 批准流程的作用

- **默认批准**：用户不说话 = 同意（24小时后自动批准）
- **降低决策负担**：大部分情况下用户不需要手动批准
- **保留控制权**：用户可以随时拒绝或自定义

### 4. 详细提案的作用

- **AI阅读**：提供完整的技术细节供AI参考
- **人类可跳过**：人类只需要读"结论先行"和"需求对齐"部分
