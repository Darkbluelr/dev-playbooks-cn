# /devbooks:delivery

Triggers Skill: `devbooks-delivery-workflow`

## Purpose

Run a change through a traceable closed-loop (Design->Plan->Trace->Verify->Implement->Archive), with clear DoD, traceability matrix and role isolation.

## Typical Use Cases

- Need complete delivery acceptance workflow
- Generate traceability matrix and DoD checks
- Ensure change package meets close-out criteria

Please invoke `devbooks-delivery-workflow` to execute the corresponding task.
