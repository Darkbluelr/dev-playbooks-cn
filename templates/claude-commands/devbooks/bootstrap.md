# /devbooks:bootstrap

Triggers Skill: `devbooks-brownfield-bootstrap`

## Purpose

Brownfield project initialization: When the current truth directory is empty, generate project profile, glossary, baseline specs and minimal verification anchors to avoid "modifying behavior while patching specs".

## Typical Use Cases

- Onboarding legacy projects to DevBooks
- Establish project glossary and baseline
- Generate initial project profile

Please invoke `devbooks-brownfield-bootstrap` to execute the corresponding task.
