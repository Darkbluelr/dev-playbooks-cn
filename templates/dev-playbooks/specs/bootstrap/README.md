# Bootstrap 规格模板

此模板用于在存量项目 Bootstrap（基线建立）阶段记录基线产物与输出清单（checklist）。

## 最小产物集合

- 项目画像（Project profile）
- 术语表（Glossary）
- 关键概念（Key concepts）
- 基线规格（Baseline specs）
- 最小验证锚点（Minimal validation anchors）

## DoR（Definition of Ready）

| 条件 | 判定 | 处理 |
| --- | --- | --- |
| 基线产物齐全 | 必须满足 | 如缺失则触发补齐流程 |
| 变更范围清晰 | 必须满足 | 如不清晰则提出澄清问题 |
| 风险标注可用 | 必须满足 | 如缺失则记录为阻塞项 |

## 验证锚点

- 输出路径：`dev-playbooks/specs/_meta/`
- 证据落点：`dev-playbooks/changes/<change-id>/evidence/`
